## Визитка в стиле плейлиста

Минималистичный лендинг под GitHub Pages, стилизованный под плейлист. Название — ваш ник, строки — кликабельные ссылки на контакты. Плавный фон и мягкий дождь на заднем плане.

### Как запустить
1) Откройте `index.html` двойным кликом — всё работает без сборки.

### Кастомизация
- Заголовок и подпись: `index.html`
  - `h1.titl` — ваш ник
  - `.auth` — подпись под заголовком
- Контакты: `index.html`
  - меняйте `href` и текст внутри блоков `.row`
- Логотип/обложка: замените `logl.jpg` (квадрат)
- Визуал и адаптив: `style.css`
  - центральная «карта» — `.app`
  - фон с переливом — `body::before`
- Дождь: `script.js`
  - плотность: `d = Math.ceil((w*h)/5200)` — меньше число → гуще
  - скорость/длина: параметры `v`, множители для `a.y`, `tx/ty`

### Структура
```
index.html     — разметка визитки
style.css      — стили, фон и адаптив
script.js      — анимированный дождь (canvas)
logl.jpg       — обложка (1:1)
```

### Публикация на GitHub Pages
1) Создайте репозиторий и загрузите файлы в корень
2) Settings → Pages → Build and deployment
3) Source: Deploy from a branch → `main` → `/` (root)
4) Сохраните и подождите 1–2 минуты

### Быстрая настройка
- Измените текст в `h1.titl` и `.auth`
- Подставьте свои ссылки в блоках `.row`
- Замените `logl.jpg` на свою картинку
